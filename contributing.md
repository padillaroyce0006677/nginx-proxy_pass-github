### Как подключить GitHub репозиторий

1. Откройте Nginx Proxy Pass GitHub
2. Нажмите **«Подключить GitHub»**
3. Авторизуйтесь через GitHub
4. Выберите репозиторий для конфигураций
5. Выберите ветку (обычно `main` или `master`)
6. Нажмите **«Синхронизировать»**

### Как создать прокси через GitHub

1. Создайте файл в репозитории: `sites-available/example.conf`
2. Используйте шаблон:
```nginx
server {
    listen 80;
    server_name example.com;
    
    location / {
        proxy_pass http://localhost:3000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }
}
```
3. Запушьте в репозиторий
4. Инструмент автоматически применит конфигурацию

### Как настроить Webhook (авто-деплой)

1. В GitHub репозитории: Settings → Webhooks
2. Add webhook:
   - **Payload URL:** `http://your-server:8080/webhook`
   - **Content type:** `application/json`
   - **Events:** Just the push event
3. Сохраните
4. При каждом пуше nginx перезагружается автоматически

### Как проверить конфигурацию

1. Нажмите **«Валидация»**
2. Инструмент запускает `nginx -t`
3. Если ошибок нет — конфигурация корректна
4. Если есть ошибки — показывает где именно
5. Исправьте и запушите заново
